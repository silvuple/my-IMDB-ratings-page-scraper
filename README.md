# my-IMDB-ratings-page-scraper
my IMDB ratings page scraper with python 3 and BeautifulSoup

## Prerequisites

* Python 3
* bs4 (Beautiful Soup - a Python library)
* pandas (Python package)
* re (Regular expression operations module of Python)
